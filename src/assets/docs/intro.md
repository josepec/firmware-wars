# Introducción a Firmware Wars

**Firmware Wars** es un wargame de miniaturas de ciencia ficción ambientado en un futuro distópico donde las corporaciones y los colectivos de supervivientes luchan por el control de los sistemas de firmware que gobiernan la civilización.

## ¿De qué va el juego?

Dos o más jugadores controlan ejércitos de miniaturas en un tablero de batalla. El objetivo varía según la misión, pero siempre implica:

- **Controlar puntos estratégicos** del mapa
- **Eliminar unidades enemigas** clave
- **Hackear sistemas** del adversario para obtener ventajas tácticas

## Componentes necesarios

| Componente | Cantidad |
|---|---|
| Miniaturas | Variable según lista |
| Dados de 6 caras (D6) | 10+ |
| Cinta métrica (pulgadas) | 1 |
| Marcadores de estado | Variable |
| Tablero de batalla | 1 (90x90 cm mínimo) |

## Escala del juego

Firmware Wars usa escala de **28mm heroico**. Las distancias se miden siempre en **pulgadas**.

## Número de jugadores

El juego está diseñado para **2 jugadores**, aunque incluye reglas opcionales para partidas de 3-4 jugadores con alianzas.
